# Data Source Platform

Data Sources are metadata-only records attached to a Project.

Supported source types:
- Google Ads
- Meta Ads
- TikTok Ads
- Snapchat Ads
- Google Analytics 4
- Shopify
- WooCommerce
- Salla
- Zid
- CSV Import
- REST API
- Webhook
- Manual Upload

The platform supports create, update, enable, disable, archive, delete, validation state, health state, sync state, connection state, and future OAuth readiness metadata.
